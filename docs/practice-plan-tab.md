# 練習計画タブ (構想メモ)

## 目的
★お気に入りに入れた技から、向こう1週間/1ヶ月の練習メニューを自動生成する。

## 必要な機能
- ★付き動画のプール
- 期間選択 (1週間 / 1ヶ月など)
- 自動メニュー生成 (頻度・順序・分散)
- 「何を何回練習したか」を俯瞰するビュー (期間別集計)
- 進捗ランクの再導入を検討 (回数閾値で自動付与、計画立案/到達度評価に使う想定)

## 現状の関連データ
- `practice` (回数)
- `practiceLog[]` (timestamps)
- `lastPracticed`
- `fav` (★)
- `RANK_DEFS` (counter.js, 0/1-4/5-14/15+ で自動導出 — 現在 vpanel から非表示)

## 保留事項
- ランクの命名 (習得中/マスター 等は一旦撤回中)
- 表示位置 (vpanel / カード / 計画タブ専用)
